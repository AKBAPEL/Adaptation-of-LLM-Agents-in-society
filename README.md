#  Адаптация LLM-Агентов в социуме

![Python](https://img.shields.io/badge/Python-3.8%2B-blue)

## Описание

Проект посвящен исследованию поведения LLM-Агентов в сложных системах с открытым взаимодействием. 
Основная цель работы — оценить возможность использования таких систем для автоматизации торгов и переговоров.



## Задачи

- Построение мультиагентной архитектуры
- Построение паттернов агентского взаимодействия
- Внедрение механизмов доверия
- Исследование эмерджентности LLM-Агентов
- Сравнительный анализ кастомных метрик качества 

## Стек

- **Язык:** Python
- **Обработка данных:** Pandas, NumPy, NetworkX (для графов)
- **Визуализация:** Matplotlib, Seaborn, Plotly
- **Агентские Фреймворки:** 
  - LangChain
  - LangGraph
  - spade-llm (custom)

##  Данные

Работа будет вестись с использованием открытых и проприетарных LLM-моделей, на синтетических данных.
